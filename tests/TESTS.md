# Orientações sobre a implementação de testes:

## DEVE

- Avaliar toda a especificação funcional e comparar com a última versão na pasta /docs/_hist/ para identificar atualizações
- Cobrir todas as definições das espeficicações, como: estrutura da página, fluxos de navegação e regras de negócio
- Navegar por acionamento de cliques nos componentes ou links

## NÃO DEVE

- Realizar a navegação por roteamento
- Buscar elementos por busca tipo "LIKE", com conteúdo parcial.
